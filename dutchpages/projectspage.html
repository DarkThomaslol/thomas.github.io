Projecten

De Java webgame

De webgame is een project dat was gebaseerd op een tutorial die mij werd aanbevolen door youtube. Destijds wist ik niets van javascript, maar ik dacht dat het cool zou zijn om een webgame te hebben die ik met vrienden kan spelen. Dus ik volgde die tutorial en sinds hij wat art online pakte, had ik zoiets van: "Hey zou het niet cool zijn als ik ook de art voor mijn spel zou maken". Maar omdat de art plaats te lang duurde om te maken, begon ik me een beetje te vervelen en stopte ermee en maakte het helaas nooit af. Ik heb de speler wel laten bewegen met WASD, maar dat is het.

Programma's gebruikt: visual studio code
Taal gebruikt: html, js

De Appformat experiment

De appformat is een project dat ik heb gemaakt, zodat ik de basis voor Windows-forms kon leren (een optie in visual studio). Tijdens het maken van dit heb ik een paar dingen geleerd, zoals het maken van een knop of hoe je de tools gebruikt die door het format zelf worden gegeven. Dit is ook de eerste project dat ik heb gemaakt na wat lessen op school, ik dacht dat dit mijn vaardigheden zou verbeteren in C#. Maar ik leerde dat dit iets heel anders was dan wat ik leerde op school, dus na het maken van een eenvoudig prototype stopte ik snel, omdat het niet zou helpen met de school lessen.

Programma's gebruikt: visual studio
Taal gebruikt: C#
